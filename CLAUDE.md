# 芯伴X1 项目上下文

## 项目简介

芯伴 X1（小芯）— 面向儿童与学生学习桌的边云协同具身 AI 陪伴机器人。

- **竞赛**: 广和通 AIoT 行业场景命题 — "AI 玩具与 AI 陪伴桌面机器人"
- **主控**: SC171 开发套件 V3
- **机械臂**: SO-ARM101 六轴桌面机械臂
- **当前阶段**: M0 硬件摸底
- **完整方案**: `芯伴X1_完整方案与Codex协作交接.md`（项目基线文档）

## 核心架构

五层智能架构：感知层 → 上下文层 → LLM 规划层 → 安全验证层 → 执行层

- 云端 LLM: DeepSeek-V4-Flash（流式 + Tool Calling）
- 端侧: Ubuntu 20.04 / Python 3.8 / Fibo AI Stack
- 机械臂通过 Arm Bridge (FastAPI) 隔离
- LeRobot 环境在独立电脑

## 注意事项

1. **不要跳过安全校验层** — LLM 不直接控制舵机
2. **环境以实机为准** — 不盲目升级系统库
3. **固定技能 + 模仿学习共存** — 保证降级方案
4. **所有性能数据由脚本测量** — 不能只写 PPT
5. **修改架构时同步更新方案文档**
6. **当前优先**: 硬件摸底、环境搭建、模块原型验证
